# valuation-backend

## Installation
Alright boyos, you will need python3, pip3 and virtualenv to get going.

On first run you need to...
```sh
mkdir <project-name>
cd <project-name>
virtualenv venv
source venv/bin/activate
pip3 install django
pip3 install djangorestframework
git clone 
```
## General Project Dev
```sh
cd <project-name>
virtualenv venv
cd <repo-clone>
python3 manage.py runserver
```
